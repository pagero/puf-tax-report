|===
|Path |Description

|`Type`
|**Type of transmission.** +
_Valid values are:_ +
`ADD` - Supplementary data added to a previously submitted report. +
`EDIT` - Correction of individual entries in a previously submitted report. +
`REPLACE_PERIOD` - Full replacement of a previously submitted report for the same period.
|===

*Example* +
_Type with example value_
[source,xml]
----
<TaxReport>
  <!-- Code omitted for clarity -->
  <Type>REPLACE_PERIOD</Type>
  <!-- Code omitted for clarity -->
</TaxReport>
----
