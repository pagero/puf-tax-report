|===
|Path |Description

|`Type`
|**Type of transmission.** +
_Valid values are:_ +
`NEW` - Initial submission for the reporting period. +
`ADDITIONAL` - Supplementary data added to a previously submitted report. +
`CORRECTIVE` - Correction of individual entries in a previously submitted report. +
`RECTIFICATION` - Full replacement of a previously submitted report for the same period.
|===
