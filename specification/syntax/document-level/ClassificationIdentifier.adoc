|===
|Path |Description

|`ClassificationIdentifier`
|**Classification Identifier. Indicates if report is considered an Income or expense report.** +
_Valid values are INCOME or EXPENSE._ +
Use `INCOME` for sales/revenue reporting (payments received from customers). +
Use `EXPENSE` for purchase reporting (payments made to suppliers).
|===

*Example* +
_ClassificationIdentifier with example value_
[source,xml]
----
<TaxReport>
  <!-- Code omitted for clarity -->
  <ClassificationIdentifier>INCOME</ClassificationIdentifier>
  <!-- Code omitted for clarity -->
</TaxReport>
----
